# Titanic-ML-Project-

# Titanic Survival Prediction Using Machine Learning

## Overview
This project is a Machine Learning Classification project based on the famous Titanic dataset. The objective is to predict whether a passenger survived or not using different machine learning classification algorithms.

The project includes data preprocessing, exploratory data analysis (EDA), data visualization, feature scaling, model training, and performance evaluation.

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Machine Learning Models
The following classification models were implemented:

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

---

## Project Workflow

### 1. Data Loading
- Imported Titanic dataset using Pandas.

### 2. Data Cleaning
- Removed duplicate values
- Handled missing values
- Dropped irrelevant columns

### 3. Exploratory Data Analysis (EDA)
- Passenger survival analysis
- Age distribution analysis
- Fare analysis
- Embarked port analysis

### 4. Data Visualization
- Count Plots
- Box Plots
- Violin Plots
- Heatmaps

### 5. Feature Scaling
- StandardScaler used for numerical feature normalization.

### 6. Model Training
- Trained multiple ML classification models.

### 7. Model Evaluation
Models were evaluated using:
- Accuracy Score
- Confusion Matrix
- Classification Report

---

## Project Objective
To build and compare machine learning classification models for predicting Titanic passenger survival using passenger information and behavioral features.

---

## Results
The models successfully predicted passenger survival with good classification performance. Random Forest and Decision Tree models showed strong prediction capability compared to Logistic Regression.

---

## Conclusion
This project demonstrates the complete machine learning workflow for a classification problem, including preprocessing, visualization, training, and evaluation of multiple ML models using Python.
